Camellya FAQ

What is the best Camellya build in Wuthering Waves?

Camellya's recommended WaveKit setup is Havoc Main DPS Build, using Red Spring, Havoc Eclipse, and Nightmare: Crownless as the main Echo target.

What stats should Camellya use?

Start with CRIT Rate or CRIT DMG / Havoc DMG / ATK%. If the rotation feels uncomfortable, improve Energy Regen or defensive comfort before chasing perfect substats.

What teams work with Camellya?

A strong reference shell is Camellya / Roccia / Shorekeeper. WaveKit can also suggest owned alternatives through the team helper.

Is Camellya beginner friendly?

Camellya's current difficulty is listed as Beginner friendly. If you are learning, pair them with safer sustain or defensive support.
